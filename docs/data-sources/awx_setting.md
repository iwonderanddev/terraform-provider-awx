# Data Source: awx_setting

Reads AWX `settings` objects.

## Example Usage

```hcl
data "awx_setting" "example" {
  id = "example"
}
```

## Argument Reference

- `id` (String, Optional) AWX object identifier used in the detail endpoint path.
